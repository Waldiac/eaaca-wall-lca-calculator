Material editor note
The material editor supports arithmetic in module fields, e.g. 10.75 + 1.48 or 8,5 + 2,25. The calculated numeric result is saved. The material editor supports both options: kg CO₂e/m³ material, kg CO₂e/m² wall area, kg CO₂e/lfdm linear product and kg CO₂e/kg material. 5. Methodology and legal disclaimer

For standard materials with the input basis kg CO₂e/m³ material, module results are converted to kg CO₂e/m² wall area by multiplication with layer thickness in metres.

Product share [%] can be used for products that cover only part of the wall area, such as timber studs in a timber-frame wall or mortar in masonry. The new Share mode defines the GWP calculation: Apply share multiplies the environmental values by the product share. Quantity includes share keeps the share visible for documentation but does not multiply GWP again, because the entered quantity already represents the real consumption per m² wall. This is recommended for mortar, reinforcement, anchors and linear profiles when quantity is entered as kg/m² or lfdm/m². Replacement cycles are shown in module B4. In this prototype, B4 is calculated as the additional product, transport/construction and end-of-life impacts of the replaced layer: B4 = replacement cycles × (A1-A3 + A4-A5 + C1 + C2 + C3/C4). Module D is shown separately and is multiplied by the total number of layer occurrences. Product share affects GWP results. For adjacent partial-area layers with the same thickness whose shares sum to 100%, the U-value calculation uses an automatically calculated equivalent λ-value for the parallel layer.

C1 and C2 are shown separately. C3/C4 is combined because a layer is normally either routed to recovery/recycling or to disposal in a scenario, not both as independent alternatives.

U-value calculation uses: U = 1 / (surface resistance + Σ d/λ). For external walls, the default is Rsi + Rse = 0.13 + 0.04 = 0.17 m²K/W. For internal or adjacent-space situations, the selectable default is 0.13 + 0.13 = 0.26 m²K/W. Layers such as CW/UW profiles may be excluded from the simplified U-value calculation while remaining included in GWP. This tool does not replace a verified EPD according to EN 15804 or a project-specific building LCA according to EN 15978.

User Manual

EAACA Wall LCA & Thermal Performance Calculator

1. Purpose of the tool

The EAACA Wall LCA Calculator is a browser-based tool for indicative assessment of wall constructions. It calculates a simplified U-value and life-cycle greenhouse gas indicators in kg CO₂e per m² wall area from user-defined material data, layer thicknesses, quantities and life-cycle module values.

The tool is intended for transparent comparison, plausibility checks, internal discussion and communication. It is not a verified environmental product declaration, not a complete building LCA and not a substitute for project-specific verification by qualified persons.

2. Basic workflow

  1. Check or create the required materials in the internal material database.
  2. Create a wall build-up by adding layers from the material list.
  3. Enter or adjust layer thickness, product share, quantity, U-value inclusion and replacement cycles in the layer table.
  4. Click Apply for a single row or Refresh / recalculate construction after several changes.
  5. Review the U-value, the module results, the layer contributions and the diagrams.
  6. Save the construction in the local construction library if it shall be reused or compared.
  7. Select saved constructions in the comparison library and generate a comparison report if required.

3. Example A: Creating a construction with newly entered materials

Use this workflow when the required material is not yet available in the database, for example a new EPD value or a manufacturer-specific dataset.

  1. Go to Add or edit material and click New material.
  2. Enter a clear material name, for example Lightweight render, project EPD.
  3. Enter the thermal conductivity λ used for the simplified U-value calculation.
  4. Select the correct GWP input basis: per m³, per m², per lfdm or per kg.
  5. Enter the module values A1-A3, A4-A5, B1, B2, B3, C1, C2, C3/C4 and D. Enter zero only where zero is intended or justified.
  6. Click Save material. The material is now available in the layer material list.
  7. Go to Wall build-up, select the new material, enter the layer thickness and click Add layer.
  8. After all layers are entered, click Refresh / recalculate construction.
  9. Enter a construction name and click Save construction to store the complete wall build-up locally.

4. Example B: Creating a construction from existing database materials

Use this workflow when the required materials are already available in the internal material database, for example AAC, calcium silicate units, mineral wool, EPS, gypsum plaster or render.

  1. Select the first material in Wall build-up, for example Gypsum plaster, enter the thickness, for example 10 mm, and click Add layer.
  2. Select the masonry material, for example AAC 400 kg/m³ or Calcium silicate unit, enter the thickness and click Add layer.
  3. Add insulation, render or other layers in the same way.
  4. Check whether every layer shall be included in the U-value calculation. Linear or mass-based components such as steel profiles may remain included in GWP but be switched off for U-value.
  5. Click Refresh / recalculate construction and review the results.
  6. Save the wall build-up under a clear construction name. If the same name is used again later, the saved construction is overwritten and the comparison table is updated.

6. Editing materials and constructions

The tool distinguishes between material master data and construction-specific layer data.

  • Material master data is edited in Add or edit material. This includes material name, λ-value, GWP input basis and module values.
  • Layer data is edited in the wall build-up table. This includes thickness, product share, quantity, U-value inclusion and replacement cycles.
  • After editing values directly in the layer table, click Apply in the row or Refresh / recalculate construction.
  • If a saved construction is loaded and then saved again under the same name, the existing construction is overwritten.
Important: Changing a material in the database affects future use of that material. For existing saved constructions, load the construction and check the values before relying on the result.

8. Product share and quantity

Product share [%] describes the share of a product in the wall area. A full-area layer normally has 100 %. For a timber-frame wall, the insulation may be entered with 85 % and the timber studs with 15 %. For masonry, the units may be entered with 98–99 % and mortar with 1–2 %. The Share mode then decides whether the product share is actually applied to GWP.

Quantity is required only for input bases per lfdm and per kg. For linear products, enter running metres per square metre wall area. For mass-based products, enter kg per square metre wall area.

9. U-value calculation and inhomogeneous layers

For normal homogeneous layers, the tool uses a simplified serial resistance model: U = 1 / (surface resistance + Σ d/λ). The selectable boundary conditions define the surface resistance, for example external wall Rsi 0.13 + Rse 0.04 = 0.17 m²K/W.

For inhomogeneous layers such as timber studs with insulation between the studs, the components should be entered as adjacent rows with the same thickness and product shares that sum to 100 %. If these conditions are met and the layers are activated for U-value, the tool automatically calculates an equivalent thermal conductivity:

λeq = share₁ × λ₁ + share₂ × λ₂ + ...

Example: 85 % mineral wool with λ = 0.035 W/(mK) and 15 % timber with λ = 0.12 W/(mK) gives: λeq = 0.85 × 0.035 + 0.15 × 0.12 = 0.04775 W/(mK). For a 160 mm layer this gives R = 0.160 / 0.04775 = 3.35 m²K/W.

Components that should influence GWP but not the simplified U-value, for example certain steel profiles or reinforcing elements, can be set to Use for U-value = No.

The automatic λeq method is a simplified area-weighted approach. It is suitable for transparent early-stage comparison, but it does not replace detailed thermal bridge or two-dimensional heat-flow assessment where required.

12. Saving, loading and comparing constructions

  • Use Save construction in the wall build-up section to store the current construction in the local construction library.
  • Use Load construction to load a saved or imported construction back into the layer table.
  • Use the checkboxes in the Comparison library to select saved variants for the comparison diagram.
  • Clear comparison removes the checkmarks from the diagram selection but does not delete saved constructions.
  • Delete selected constructions removes the selected constructions from the local construction library.

13. Report and PDF output

The central Report / PDF output section is used for print and PDF generation. Select the desired report content and click Generate selected report preview / PDF.

The report preview appears directly below the report control. Use the print button in the report preview or the browser print dialog to print or save as PDF. Microsoft Edge or Google Chrome are recommended for PDF output. Before printing, check page orientation and scaling; comparison reports with many constructions may be easier to read in landscape orientation.

14. Quality checks before using results

  • Check that all material data comes from the intended source and that the declared basis is correct.
  • Check whether values entered as zero are intended zero values or missing data.
  • Check thicknesses, product shares, quantities and U-value activation for every layer.
  • Check that inhomogeneous layers are entered as adjacent rows with equal thickness and shares summing to 100 % if automatic λeq shall be used.
  • Do not compare constructions with different functional requirements without explaining the differences.
  • Document assumptions, sources and deviations from verified EPD or project data.
