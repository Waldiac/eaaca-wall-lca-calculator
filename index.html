4. Add or edit material

The material editor supports both options: kg CO₂e/m³ material and kg CO₂e/m² wall area. If kg CO₂e/m³ is selected, the GWP result is multiplied by layer thickness in metres. If kg CO₂e/m² is selected, the GWP values are used directly.

5. Methodology / Disclaimer

For standard materials with the input basis kg CO₂e/m³ material, module results are converted to kg CO₂e/m² wall area by multiplication with layer thickness in metres.

For individual project-specific materials with the input basis kg CO₂e/m² wall area, module results are used directly. Thickness changes then affect the U-value, but not the GWP of that material.

U-value calculation uses: U = 1 / (Rsi + Σ d/λ + Rse). For this prototype, Rsi = 0.13 m²K/W and Rse = 0.04 m²K/W are used for external walls.

Module D is shown separately and is not automatically included in the A-C result. This tool does not replace a verified EPD according to EN 15804 or a project-specific building LCA according to EN 15978.
