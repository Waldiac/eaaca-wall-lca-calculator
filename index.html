4. Add or edit material

For newly added individual materials, the GWP values A1-A3 to D shall be entered as kg CO₂e/m² wall area, unless the basis is deliberately changed to kg CO₂e/m³ material. If “kg CO₂e/m² wall area” is selected, the values are used directly and are not multiplied by thickness.

5. Methodology / Disclaimer

For standard materials, GWP values are treated as kg CO₂e per m³ material. The result per m² wall area is calculated by multiplying the material value by the layer thickness in metres.

For individual project-specific materials with the basis “kg CO₂e/m² wall area”, the GWP values are used directly. Thickness changes then affect the U-value, but not the GWP of that material.

U-value calculation uses: U = 1 / (Rsi + Σ d/λ + Rse). For this prototype, Rsi = 0.13 m²K/W and Rse = 0.04 m²K/W are used for external walls.

Module D is shown separately and is not automatically included in the A-C result. This tool does not replace a verified EPD according to EN 15804 or a project-specific building LCA according to EN 15978.
