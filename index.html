Wall LCA & Thermal Performance Calculator

Indicative calculation of U-value and GWP per m² wall area for transparent comparison of AAC wall assemblies and alternative constructions.

In accordance with Directive (EU) 2024/1275 (EPBD recast): this tool supports transparent communication of life-cycle greenhouse gas indicators for wall constructions. Results are indicative and depend on selected datasets and scenarios.

4. Add or edit material

The material editor supports both options: kg CO₂e/m³ material, kg CO₂e/m² wall area and kg CO₂e/m linear product. If kg CO₂e/m³ is selected, the GWP result is multiplied by layer thickness in metres. If kg CO₂e/m² is selected, the GWP values are used directly. If kg CO₂e/m is selected, the GWP values are multiplied by the layer's linear amount in m/m². B4 is calculated from the replacement cycles of each layer.

5. Methodology / Disclaimer

For standard materials with the input basis kg CO₂e/m³ material, module results are converted to kg CO₂e/m² wall area by multiplication with layer thickness in metres.

Product share [%] can be used for products that cover only part of the wall area, such as timber studs in a timber-frame wall. The share factor multiplies the environmental module values of the layer. For linear products with input basis kg CO₂e/m, the values are multiplied by the layer's Linear amount [m/m²]. Replacement cycles are shown in module B4. In this prototype, B4 is calculated as the additional product, transport/construction and end-of-life impacts of the replaced layer: B4 = replacement cycles × (A1-A3 + A4-A5 + C1 + C2 + C3/C4). Module D is shown separately and is multiplied by the total number of layer occurrences. Product share affects GWP results. The U-value calculation remains a simplified layer-by-layer calculation and does not fully model parallel heat flow paths.

C1 and C2 are shown separately. C3/C4 is combined because a layer is normally either routed to recovery/recycling or to disposal in a scenario, not both as independent alternatives.

U-value calculation uses: U = 1 / (surface resistance + Σ d/λ). For external walls, the default is Rsi + Rse = 0.13 + 0.04 = 0.17 m²K/W. For internal or adjacent-space situations, the selectable default is 0.13 + 0.13 = 0.26 m²K/W. Layers such as CW/UW profiles may be excluded from the simplified U-value calculation while remaining included in GWP. This tool does not replace a verified EPD according to EN 15804 or a project-specific building LCA according to EN 15978.

User Manual

EAACA Wall LCA & Thermal Performance Calculator

1. Purpose of the tool

The EAACA Wall LCA Calculator is a browser-based tool for estimating the thermal performance and global warming potential (GWP) of wall constructions from user-entered material data. This version does not rely on an external material database. All relevant material values are entered and maintained directly by the user within the tool.

The calculator is intended for transparent comparisons of construction variants, plausibility checks and communication purposes. It is not a substitute for a verified environmental product declaration according to EN 15804, a complete building LCA according to EN 15978 or a formal regulatory assessment.

2. Recommended workflow

  1. Open the calculator in the browser.
  2. Enter or edit the required materials in the internal material database.
  3. Create the wall construction by adding one layer per material.
  4. Enter the thickness of each layer in millimetres.
  5. Use Apply for a single row or Refresh / recalculate construction after editing several rows.
  6. Review the U-value, the layer-by-layer GWP result and the total result.
  7. Save constructions locally and compare up to five variants.
  8. Generate a report preview and print or save it as PDF if required.

6. Creating a wall construction

A wall construction is entered as a sequence of layers. Each row normally represents one physical layer, for example internal plaster, masonry, insulation and external render.

  1. Select a material from the internal material list.
  2. Enter the layer thickness in millimetres.
  3. Click Add layer.
  4. Repeat this process for all layers in the desired order.
  5. Edit product share, linear amount, U-value inclusion and replacement cycles directly in the layer table where required.
  6. Click Apply in the row or Refresh / recalculate construction.

7. Editing existing entries

The tool distinguishes between material master data and construction-specific layer data.

  • Edit material master data in the Add or edit material section.
  • Edit construction-specific values, such as layer thickness or product share, in the construction table.
  • Use the row Apply button to make a single row change effective.
  • Use Refresh / recalculate construction after editing several rows.
  • Check the result table again after each relevant change.

10. Replacement cycles

Replacement cycles define additional replacements within the reference period. In this prototype, B4 is calculated as: B4 = replacement cycles × (A1-A3 + A4-A5 + C1 + C2 + C3/C4).

ValueInterpretationTypical use
0No replacementService life approximately 50 years or longer.
1One replacementService life below the reference period, e.g. around 40 years.
2Two replacementsShorter-life components, e.g. around 25 years.

11. Saving and comparing constructions

Saved constructions are stored locally in the user's browser. Other users do not see these saved variants. Up to five saved constructions can be selected for the comparison diagram.

12. Reports, PDF and HTML export

  • Generate construction report preview / PDF creates a report for the current construction.
  • Generate comparison report preview / PDF creates a comparison report.
  • The browser print dialog can be used to print or save the report as PDF.
  • Microsoft Edge or Google Chrome are recommended for PDF output.

13. Quality checks before using results

  • Check that material names are understandable and not duplicated.
  • Check that thickness values are realistic and entered in millimetres.
  • Check that the environmental data basis is consistent with the quantity calculation.
  • Check whether zero values mean “not relevant”, “not available” or “not yet entered”.
  • Document the data sources used for the material values.
  • Do not compare variants with different functional requirements without explaining the difference.

14. Troubleshooting

IssuePossible solution
Material list is emptyAdd materials in the internal material editor or reset the browser's local storage if an old test version is still cached.
Long names overlapUse shorter English material names and avoid combined German/English labels.
Changed layer values do not affect the resultClick Apply in the edited row or Refresh / recalculate construction.
Unexpected GWP valuesCheck whether the input basis is per m³, per m² or per metre and whether the layer thickness, product share or linear amount is correct.
PDF export does not workUse the browser print function in Microsoft Edge or Google Chrome.

15. Important limitation

The calculator depends on the quality and consistency of the data entered by the user. Results should be treated as indicative unless all material data, conversion rules and calculation assumptions have been checked and documented.
